# Railway uchun tayyor paket

1. GitHub'da yangi public yoki private repo oching.
2. Shu papkadagi barcha fayllarni o'sha repo'ga yuklang.
3. Railway'da `Connect Repo` ni bosing va shu repo'ni ulang.
4. `Variables` bo'limiga shularni kiriting:
   - `BOT_TOKEN=YANGI_TOKEN`
   - `ADMIN_USERNAME=abdullayevv_tm`
   - `ADMIN_USER_ID=` (ixtiyoriy, botga /start yozgandan keyin saqlaydi)
5. Deploy qiling.
6. Telegram'da botga `/start` yozing.
7. Botni guruhga qo'shing, admin qiling, BotFather'da Privacy OFF qiling.

Agar deploy xato bersa, `Deployments` logini ochib xatoni yuboring.
